How to import into Foundry

Open the Items menu.

Click "Create Item" then select the item type as "Weapon". You can close the item preview which opens.

Right click the new item then click "Import Data".

Select the JSON file exported from Armoury. The name and image of the item should change. If it didn't, something went wrong with the generation.

Drag the finished gun into the "Guns" folder. Then open your character sheet's inventory and drag the gun across to equip it.
